<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

  - npm

### Installation

```sh
git clone https://github.com/iLLoReal/spotify-list
```
Ou par le biais de l'interface github: download ZIP > extraire à la racine de votre dosser web.
```sh
cd spotify-list
npm install
npm run dev
```
